# Scripts

This directory contains scripts that are used to **generate data files** for the server. 
All **path references are relative to the script** unless otherwise specified. 
**All scripts output to the '../resources' directory.**

## Requirements

Scripts use the resources extracted from the minecraft server jar. (*Obviously gitignored*) 
[How to extract](https://minecraft.wiki/w/Minecraft_Wiki:Projects/wiki.vg_merge/Data_Generators)

Place the generated minecraft folder in './resources'.

Requirements:
- Python 3.12
